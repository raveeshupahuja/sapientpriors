# Machine Learning Engineer Intern

## About Sapient Priors

We're building the future of AI agents that truly learn. At Sapient Priors, we're working towards completely autonomous agents that learn continuously from their environment, adapt to new challenges, and improve over time without human intervention. Our first step is building Personalization as a Service - a Continuous Learning API that transforms static LLM applications into agents that learn and improve with every interaction.

Our team includes ML veterans with 12+ years of experience from Microsoft, Twitter, and Dropbox, alongside product management leaders from Uber and Amazon. We're creating foundational infrastructure that enables AI systems to break free from the limitations of static models and deliver experiences that get better over time.

Learn more at: https://www.sapientpriors.com

**Evaluation & Testing Infrastructure**

**Production Engineering**
* Deploy features to production with proper monitoring and observability
* Build reliable and scalable services for real-time learning and inference
* Implement logging and debugging tools for production learning systems
* Ensure high availability and performance of the Continuous Learning API
* Create documentation and runbooks for your services
